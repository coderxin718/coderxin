关于我

一段从代码到智能的旅程。

我写 Java 后端，也折腾大模型。对我来说，最有趣的事情不是调通一个 API，而是让一个 Agent 自己决定"下一步该调哪个工具" — 并且结果真的可信、可追溯。

在托普云农实习期间，我从零搭建了一个面向农业场景的智能问答系统：从文档切分策略到向量检索，从 ReAct Agent 编排到 SSE 流式输出，从幻觉控制的四道防线到 MCP 动态工具扩展。那种把一整套 AI 链路跑通的感觉，让我确信这就是我想深耕的方向。

我关注接口的稳定性、系统的可观测性和回答的可追溯性。在垂直领域把 AI 真正落地，而不是停留在 Demo — 这大概就是我的工作哲学。
