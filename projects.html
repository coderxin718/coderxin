AI·Agent

云农智能体

让 Agent 自己决定该查什么、怎么查。这是一个面向农业的智能问答系统，核心亮点是 ReAct Agent 多工具编排 + 向量/关键词混合检索 + 四道幻觉防线。用户拍一张叶片照片，系统自动完成从图像分析到诊断报告的完整推理链 — 带来源引用、可追溯。
计算机视觉·全栈

植物表型智能分析平台

用代码"看"懂作物。这个平台打通了从田间多模态数据采集（RGB/高光谱/LiDAR/热红外）到云端在线分析的完整链路：30+ 植被指数实时计算、YOLOv8 植株检测、LiDAR 点云三维重建、轻量化病害识别模型边缘部署。不只是 Demo — 已在实际的小麦和茶树试验中投入使用。
微服务·实时通信

千言 InfiniteChat

从零手写一个 IM 系统是什么体验？7 个微服务、Netty 自建 WebSocket 协议、Kafka 异步解耦、分布式精准推送 — 这个项目是我对后端工程能力的系统训练。支持单聊、群组、朋友圈、红包，跑在 Docker 上一键部署。它让我真正理解了微服务拆分、消息可靠性和容器化运维。
